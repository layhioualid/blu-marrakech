# Images

Le site utilise actuellement des visuels Unsplash distants configurés dans `next.config.ts`.
Ajoutez ici les médias officiels de BLU Marrakech, puis remplacez les URLs dans `lib/constants.ts` et les composants concernés.

Pour activer la vidéo de couverture, copiez `.env.example` vers `.env.local` et renseignez
`NEXT_PUBLIC_HERO_VIDEO_URL` avec une vidéo MP4/WebM optimisée et hébergée.
